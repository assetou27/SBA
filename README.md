# Fruits and Vegetables Website

# Description
" 3 Responsive pages to promote the benefits of eating fruits and vegetables. It talks about nutritional information, give tips and opportunities to connect with the community.

# Features
- **Homepage:* Includes tips and a table of fruit benefits.
- **Contact Us:*  Allows users to send messages and find contact details.
- **About Us:* Provides information about the team and mission.

# Technical Specifications
- **HTML:** Includes a table, forms, dropdown menu, and responsive layout.
- **CSS:** Uses inline, internal, and external styling. Includes two animations (`pulse` and `fadeIn`).
- **Framework:** Bootstrap for layout and styling.
